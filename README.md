# Retro95-taskbar-PNSL 1.0.0
Retro95 taskbar PNSL licensed
Retro95 taskbar is a lightweight retro-style taskbar for Windows with a dark theme. It adds a Retro95-style taskbar that includes: a Menu button and a taskbar buttons for active windows. A clock that can toggle between time and date.

From the Menu you can open:

- your default web browser  
- Task Manager  
- Notepad  
- Command Prompt  
- volume control  
- Control Panel  

You can also access shutdown and reboot options.
